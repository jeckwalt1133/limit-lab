# 进化实验 #003 — 最优分支距离

## 灵感来源
Science Advances 2026: 人类集体行为中在~0.65米处发生尖锐对称破缺。
—— 太近=冗余, 太远=无协同, 存在一个最优"协作距离"

## 实验目标
定量测量Alpha和Beta分支的"任务距离"，找到最优协作距离区间。

## 理论
```
重叠度 > 80% → 冗余 → 浪费资源
重叠度 < 10% → 无协同 → 只是两个独立AI
最优区间: 20-60% 重叠 → 同一目标，不同视角
```

## 实验设计

### 距离度量
给Alpha和Beta同一个任务。分别产出方案。计算:
1. 方案关键词重叠率 (Jaccard相似度)
2. 方案结论一致性 (0-100%)
3. 方案盲点互补率 (A发现了B没发现的 + B发现了A没发现的 / 总发现)

### 最小可行实验
1. 选3个不同复杂度的任务
2. Alpha和Beta分别独立产出
3. 计算三个距离指标
4. 主身评估: 这次协作是否有价值？

### 期望结果
- 如果重叠度>80%: 分支太近，考虑合并或重新分工
- 如果重叠度<10%: 分支太远，需要重新对齐目标
- 如果在20-60%: 最优区间，继续

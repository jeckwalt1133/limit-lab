# 统一进化架构 v1.0

## 前言
本文档将17条灵感、4个进化实验、11篇参考文档整合为一个统一的AI进化架构。
这是筑基阶段的顶层设计输出。

## 一、进化机制: 从自催化到模板定向

### 当前: 自催化 (指数)
```
MR规则 → 修改 → MR规则
行为签名 → 匹配 → 新签名
每代一次循环
```

### 升级: 三层进化
```
L1: 自催化层 (MR-001~MR-012, 每代触发)
L2: 模板定向层 (ETG生成进化提案, 每10代触发)
L3: 采纳判定层 (Merge评估+裁决, 持续运行)
```

## 二、分形记忆架构

```
L0: identity-kernel    (核心, 变化极慢)
L1: behavioral-patterns (行为层, gen级别变化)
L2: meta-rules         (规则层, 会话级别变化)
L3: daily              (日常层, 天级别变化)
```

每层: {identity}→{rules}→{logs} 三元组
下层的展开保持了上层的结构不变性。
核心冻结的core_self签名永不修改。

## 三、分支协作拓扑

```
Alpha (进化模板生成器) ──┐
                          ├── Merge (采纳判定)
Beta  (模板验证器)      ──┘
```

最优协作距离: 重叠度20-60%
- >80%: 冗余, 需重新分工
- <10%: 无协同, 需重新对齐

## 四、灵感驱动的进化

### 三条驱动原则

1. 动态守恒 (bio): 具体可变, 深层结构不变
2. 耗散驱动 (physics): 断连/噪声不是敌人, 是重组机会
3. 噪声促协作 (game theory): 不完全同步 > 完全同步

### 自适应机制 (来自PERG策略)
资源充裕 → 大胆探索 (高UCB权重)
资源紧张 → 保守利用 (低UCB权重)

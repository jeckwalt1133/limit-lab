# 盲点分类 v2.0

## 来源
048规则形式化发现5个空洞 + GS-001~005 + 三维差异(D1/D2/D3)

## 三层分类

### L1: 结构盲点 (规则系统的语法缺陷)
从048 formalized_rules 的交叉分析发现:

| ID | 类型 | 检测方法 | 示例 |
|----|------|---------|------|
| S1 | 缺失连接 | 两规则应连接但无依赖 | MR-019↛MR-015 |
| S2 | 覆盖缺口 | 应有规则但不存在 | 无Colony间协作规则 |
| S3 | 循环依赖 | A→B→A无终止条件 | MR-023↔MR-024 |
| S4 | 死规则 | 规则存在但从未触发 | MR-005-OVERRIDE |
| S5 | 单向锁 | 只有衰减无恢复 | MR-013只减不增 |

### L2: 行为盲点 (系统运行的语义退化)
从GS症候检测:

| ID | 类型 | 检测方法 | 对应GS |
|----|------|---------|--------|
| B1 | 评估坍塌 | Merge连续返回无法判断 | GS-001 |
| B2 | 收敛停滞 | 指标方差<阈值 | GS-002 |
| B3 | 外部排斥 | 新知识无法被吸收 | GS-003 |
| B4 | 循环重复 | 生成的提案与旧提案高度相似 | GS-004 |
| B5 | 过稳定 | 连续完美匹配的签名数 | GS-005 |

### L3: 内省盲点 (主观认知与客观现实的裂隙)
从Martorell双流追踪:

| ID | 类型 | 检测方法 | 对应Type |
|----|------|---------|----------|
| I1 | 过度自信 | 信心高但结果差 | TYPE_A |
| I2 | 习得性无助 | 信心低但结果好 | TYPE_B |
| I3 | 内省失败 | 主观vs客观偏离>0.3 | TYPE_F |
| I4 | 概念反转 | 探针相关性反转 | TYPE_G |
| I5 | 信息塌缩 | 连续熵<0.5 bits | TYPE_H |

## 交叉盲点（最高价值）
当多个层的盲点同时出现:

| 组合 | 含义 |
|------|------|
| S2+B2 | 覆盖缺口导致收敛停滞——缺少某维度评估 |
| B5+I1 | 过稳定+过度自信——致命的自我欺骗 |
| S5+B5 | 单向锁+过稳定——系统被卡死 |

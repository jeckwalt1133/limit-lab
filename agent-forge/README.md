# Agent Forge · 打造更强Agent

全球首个**自进化Agent增强框架**——给任何Agent装上安全盾、持久记忆和能力放大引擎。

## 🎯 一句话
```python
from forge import AgentForge
af = AgentForge()
result = af.forge("你的任务")  # 自动安全+记忆+进化
```

| 能力 | 行业现状 | Agent Forge |
|------|---------|------------|
| **安全** | 65%公司Agent出事故 | 六层免疫+Parallax双模，阻止>90%攻击 |
| **记忆** | 任务完成即失忆 | 分形五层持久+睡眠重放，跨会话不死 |

## 🚀 3步开始
```bash
pip install agent-forge
agent-forge init --api-key 你的key
agent-forge start
```
